Privacy Policy

Effective date: March 10, 2026 · Last updated: March 10, 2026

The short version: Sprout Math does not collect, transmit, or store any personal information on a server. There are no accounts, no analytics, no ads, no in-app purchases, and no network calls. Everything stays on the device.

1. Information We Collect

None. Sprout Math does not collect personal information from children or adults. Specifically:

  • No name, email, phone number, or account credentials are collected or transmitted.
  • No device identifiers, advertising IDs, or tracking data are collected.
  • No GPS location, IP address, or geographic data is accessed.
  • No photos, contacts, microphone input, camera, or other device sensors are accessed.
  • No usage analytics, screen views, tap events, or behavioral data are collected.
  • No crash reports are transmitted to external services.

2. Data Stored on the Device

Sprout Math stores learning progress locally on the device using Apple's Core Data framework within the app's private sandbox. This includes:

  • A child profile containing only a first name (entered by the parent or child) and a locally generated identifier.
  • Session history, mastery progress, review schedules, and sticker records.
  • App preferences such as selected theme, companion character, and narration settings.

This data never leaves the device. It is not synced to any cloud service, server, or third party. Deleting the app permanently removes all locally stored data.

3. Network Activity

Sprout Math makes zero network requests in version 1. The app is designed to function entirely offline. There is no backend server, no API, no analytics endpoint, and no telemetry. All content (questions, lessons, hints, and rewards) is bundled inside the app.

4. Third-Party Services and SDKs

Sprout Math does not integrate any third-party SDKs, analytics services, advertising networks, or social media frameworks. The only frameworks used are Apple's own: SwiftUI, Core Data, and AVFoundation (for text-to-speech narration).

To be explicit, Sprout Math version 1 does not use: Sign in with Apple or any other sign-in mechanism; push notifications; in-app purchases or subscriptions; third-party analytics or crash reporting services; advertising networks; or social media frameworks.

5. Children's Privacy (COPPA Compliance)

Sprout Math is designed for children ages 5 through 11 and is intended for listing in the Apple App Store Kids Category (Ages 6-8).

  • The app does not collect personal information from children as defined under the Children's Online Privacy Protection Act (COPPA).
  • No data is transmitted to any server, so no parental consent mechanism is required for data collection.
  • The app contains no links to external websites, no social features, no messaging, and no user-generated content.
  • Settings and controls that affect the learning experience are gated behind a parent verification challenge (a math problem a young child is unlikely to solve).

Note: Apple may collect certain anonymized, aggregated data at the operating system level (such as crash logs) if a user has opted into this at the device level through iOS settings. This data is governed by Apple's Privacy Policy (apple.com/privacy) and is separate from Sprout Math's practices. Sprout Math itself initiates no data collection of any kind.

6. Diagnostics Export

Parents can optionally export a local diagnostics file from the parent dashboard. This file contains session history and placement data for debugging purposes. The export stays on the device as a local file and is never transmitted automatically. The parent controls whether to share, save, or delete this file.

7. Data Security

All data is stored in the app's sandboxed container using Apple's Core Data framework. The operating system prevents other apps from accessing this data. Since no data is transmitted over a network, there is no risk of interception or server-side breach.

8. Future Versions

If a future version of Sprout Math introduces cloud sync, accounts, or any form of data transmission, this privacy policy will be updated before that version ships. Any such feature would require explicit opt-in from the parent and would be clearly disclosed.

9. Your Choices

  • Delete all data: Uninstall the app. All locally stored progress, profiles, and preferences are permanently removed.
  • Review stored data: Use the parent dashboard and diagnostics export to see what the app has recorded.
  • No account to close: There is no account, so there is nothing to cancel or deactivate.
